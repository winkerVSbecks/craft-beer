## Getting Started
```bash
$ git clone https://github.com/rangle/react-redux-seed
$ npm install
```

## npm scripts

### Dev
```bash
$ npm run dev
```

Open [localhost:3000](http://localhost:3000) in your browser.

### Tests

#### Single Run
```bash
$ npm run test
```

#### Watch Files
```bash
$ npm run test:watch
```

#### Coverage
```bash
$ npm run cover
```

### Production
```bash
$ npm run start
```
